##This project is a hybrid spam email detection system developed using Python, combining machine learning and rule-based techniques with a graphical user interface. It uses a dataset of messages to train a Naive Bayes model with TF-IDF vectorization for accurate spam classification. In addition, a rule-based approach checks for common spam indicators such as suspicious keywords, generic greetings, links, and excessive punctuation. The system integrates both methods to improve prediction reliability. A user-friendly GUI built with Tkinter allows users to input messages and instantly check whether they are spam or not. This project demonstrates practical implementation of machine learning, text processing, and interactive application development.
